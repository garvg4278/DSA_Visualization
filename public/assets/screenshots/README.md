## How to Capture

### GIFs
- Use [LICEcap](https://www.cockos.com/licecap/) (Windows/Mac)
- Or [Peek](https://github.com/phw/peek) (Linux)
- Or [ScreenToGif](https://www.screentogif.com/) (Windows)

### Screenshots
- Chrome DevTools → toggle device toolbar for mobile screenshots
- Use `Cmd+Shift+4` (Mac) or `Win+Shift+S` (Windows)
